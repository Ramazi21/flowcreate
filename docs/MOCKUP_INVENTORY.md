# Инвентаризация макетов и экранов

Локальные файлы макетов (PNG/PDF и т.д.) в рабочей папке **не обнаружены**. Ниже — базовая карта экранов, выровненная по референсу [SiteGovna](../reference/SiteGovna/README.md), как временная спецификация до появления ваших макетов.

## Маршруты (desktop-first, адаптив ожидается по макетам)

| Маршрут | Назначение | Состояния / примечания |
|---------|------------|-------------------------|
| `/` | Главная: герой, категории, новинки, блок дизайнеров | Скролл к `#collections`, `#about`, `#contacts` |
| `/catalog` | Каталог товаров | Фильтр по категории через `?category=` (план) |
| `/designers` | Список дизайнеров | Карточки студий |
| `/designers/[slug]` | Профиль дизайнера | Динамический slug |
| `/cart` | Корзина | Пока статичный макет без сервера |
| `/login` | Вход | Форма-заглушка без авторизации |

## Брейкпоинты

До получения макетов используются типовые Tailwind-срезы: `sm` 640px, `md` 768px, `lg` 1024px. **Замените** на значения из макетов, когда файлы будут доступны.

## Дизайн-токены (временные)

Взяты из референса как стартовая палитра до сверки с макетами:

- `cream` #F9F5F0, `sand` #EDE6DC, `terracotta` #B36B4D, `charcoal` #2C2622
- Шрифты: Cormorant Garamond (заголовки), Noto Sans (текст) — см. `src/app/layout.tsx`

## Что добавить, когда появятся макеты

1. Путь к папке с экспортами (или Figma-ссылку).
2. Список отличающихся экранов (чекаут, модалки, пустые состояния).
3. Точные отступы, сетка, компонентная библиотека.
