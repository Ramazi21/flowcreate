# Референс SiteGovna (Atelier Local)

Проект распакован из архива `SiteGovna.rar`. Папка `.next` удалена для экономии места; при необходимости выполните `npm run build` внутри `SiteGovna`.

## Быстрый запуск

В PowerShell из каталога `SiteGovna`:

```powershell
.\run-local.ps1
```

Или вручную: см. [SiteGovna/README.md](SiteGovna/README.md).

**Не коммитьте** файл `SiteGovna/.env` с секретами.
